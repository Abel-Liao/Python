1.
回调函数=>有传递有回归
 def test(n):
  return n*test(n-1)
 test(5)  求5的阶乘
2.
global是让局部变量使用全局变量
 a = 10
 def test():
  global a
  a = 11
  print(a) => 11
 print(a) => 11
nonlocal是让内部函数使用外部函数
 a = 10
 def test():
  a = 11
  print(a) => 11
  def test1():
   nonlocal a
   a = 12
   print(a) => 12
  test1()
 test()
 print(a) => 10
 
注意 
1.nonlocal 只适用于局部变量
2.python没有会计作用域
 e.g:
  if true:
   a = 11
  print(a) => 11
